# ✈️ Flight Booking System

A RESTful Flight Booking System developed using Java, Spring Boot, Spring Data JPA, Hibernate, and PostgreSQL.

## 🚀 Features

- Flight Management
- Booking Management
- Passenger Management
- Payment Management
- CRUD Operations
- Pagination & Sorting
- Exception Handling
- Layered Architecture
- REST APIs

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven

## 📌 Project Highlights

- Developed REST APIs for Flight, Booking, Passenger and Payment modules.
- Implemented CRUD operations, pagination, sorting and custom exception handling.
- Built business logic for seat availability, booking cancellation and payment management.
- Used Spring Data JPA with PostgreSQL for persistent data storage.

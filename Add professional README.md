# ✈️ Flight Booking System

A RESTful Flight Booking System developed using Java, Spring Boot, Spring Data JPA, Hibernate, and PostgreSQL.

## 🚀 Features

- Flight Management
- Booking Management
- Passenger Management
- Payment Management
- CRUD Operations
- Pagination & Sorting
- Exception Handling
- Layered Architecture
- REST APIs

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven

## 📌 Project Highlights

- Developed REST APIs for Flight, Booking, Passenger and Payment modules.
- Implemented business logic such as seat availability and booking cancellation.
- Used Spring Data JPA with PostgreSQL for database operations.
- Implemented custom exception handling using Global Exception Handler.
